# common-project
Spring MVC 구조 기반 도서 관리 시스템

<br/>

## 1. 제작 기간
 - 2023.03.18 ~ 2023.03.24

<br/>

## 2. 개발 환경 버전
 - JDK 11
 - eclipse 2020-09
 - Tomcat 8.5.85

<br/>

## 3. 실행 방법
 - Tomcat과 eclipse 연동 후 Tomcat 서버 실행 
 - http://localhost:8080/home 접속

<br/>

## 4. 구조
### frontend
 - 사용 기술 
   - JSP
   - CSS
   - JavaScript
   - JQuery
 - 파일 위치 
   - .jsp : /backend/src/main/webapp/WEB-INF/views/
   - .css, .js : /backend/src/main/webapp/resources/
 
<br/>

### backend
 - 사용 기술
   - Spring
   - Maven
   - MySQL
   - Tomcat 
 - 파일 위치
   - spring : /backend/
